## Git merge

Предположим, что во время работы наша feature-ветвь "отстала" от базовой ветви:

```
            (C) - (D)
          /
(A) - (B) - (E)
```

Для того, чтобы вновь сделать историю изменений линейной, существует два возможных метода работы:
- git merge
- git rebase

Рассмотрим `git merge` подробнее.
